Sentiment Classification of Movie Reviews

Exploring NLP-driven techniques to classify movie reviews into sentiments using engineered features and machine learning models.

Project Overview

This project uses natural language processing techniques and classifiers to analyze and predict sentiment in movie reviews. Using various engineered features such as unigrams, bigrams, POS tags, and lexicon-based attributes, it evaluates sentiment across multiple ML models.

Dataset

The dataset was sourced from a Kaggle competition based on Rotten Tomatoes reviews. It includes annotated sentiment labels from 0 (Negative) to 4 (Positive) and was split into training and testing sets. Kaggle Dataset Link

Key Visualizations

Sentiment Distribution

Displays frequency of sentiment labels. Neutral sentiments appear most frequently in the dataset.

Word Frequency

Highlights the most common words across reviews. 'Movie', 'make', and 'film' are highly recurring terms.

Word Length Distribution

This histogram shows the frequency of word lengths in the dataset. Shorter words (2–5 characters) occur most often in movie review phrases.

Observation & Conclusion

Classifiers such as Random Forest and SVM performed best using LIWC and combined SL+LIWC feature sets. Preprocessing impacted results differently across models, showing that unfiltered data sometimes yielded better accuracy. The best result achieved was 56% accuracy.
